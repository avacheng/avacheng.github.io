.Net String.Format 整理

2019-07-07 Sunday
.Net

日期

  • 使用系統設定的 DateTimeFormatInfo.DataSeparator 分隔年月日 ( 黑暗執行緒 )

    Console.WriteLine(string.Format("{0:yyyy/MM/dd}", DateTime.Today));

  • 強制使用 “/” 分隔 → 加上 Escape 字符 “\” ( 黑暗執行緒 )

    Console.WriteLine(string.Format("{0:yyyy\\/MM\\/dd}", DateTime.Today));

Azure 介面簡介

2019-10-06 Sunday
azure / microsoft / cloud

就是對 Azure 入口網站（Azure Portal）做個簡單的介紹…

網址：https://portal.azure.com/

介面簡介

登入後的畫面如下。左邊是功能清單，分為固定和自訂上下兩個區塊（1 和 2），區塊上方的《 符號可以將功能清單縮小，只顯示小圖示。畫面上方是輔助與提示區塊（3 和 4）。中間白色區塊是主要工作區，資源（指 Azure 提供的服務項目）資訊的顯示與操作會在此進行；首頁是顯示預設的儀表板，包含服務項目與最近資源等等，供使用者登入後快速存取（5 - 8）。

  1. 功能清單：固定項目，包含建立新資源、回到首頁、到自訂的儀表板或列出所有服務
  2. 我的最愛：自訂項目，特定資源項目群的捷徑，可透過其上方區塊 1 的「所有服務」來進行設定（如下圖）
  3. 輔助工具區：包含帳戶切換、工作提示、客戶支援等等。
  4. 帳戶：顯示目前帳戶、登出帳戶
  5. Azure 服務：顯示熱門的資源項目
  6. Microsoft Learn：連結到微軟針對 Azure 開的線上課程
  7. 基礎結構管理與成本管理的捷徑
  8. 最近的資源：快速連結到最近使用的資源
